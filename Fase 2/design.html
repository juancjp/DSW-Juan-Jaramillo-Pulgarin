El lenguje de marcado de hipertexto o (HTML), es un código basado en una serie de etiquetas que se utilza para los sitios Web, estas etiquetas se organizan de tan manera que permite estructurar en segmentos diferentes partes del sitio.

Este lenguaje a su vez, se basa en una serie de etiquetas de apertura y una de cierre a excepción de algunas que no tienen etiqueta de cierre, considerados como elementos vacios, por ejemplo la etiquete "img" la cual no tiene etiqueta de cierre, tambien la etiqueta "br", entre otras.
